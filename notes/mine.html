Mine and scrape MARC posts from the Internet Archive

2019-04-03

Programming Historian is a great resource for learning digital methods. In a particularly interesting lecture, one can learn how to scrape MARC posts from the Internet Archive using Python.

This is especially useful since libraries such as British Library, Library of Congress and Boston Public Library has uploaded large amounts of scanned, OCR:ed documents as well as its associated metadata, including MARC/XML.

The purpose of the lecture is that download several MARC posts from a chosen collection using a script. The posts can then be scraped using another script in order to extract certain fields and indicators from the chosen collection. This extracted data can then be visualised, for example as a tag cloud.

For instance, one may scrape MARC field 260 and indicator a on a collection of posts in order to examine in which geographic places a set of documents has been published.

Since the code of the lecture is written for Python 2, I have translated it to Python 3. A Github repo can be found here.
